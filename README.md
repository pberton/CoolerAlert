Setup
=========
Install NodeJS on RaspberryPi (Raspbian)

sudo apt-get update;
sudo apt-get upgrade;

wget http://node-arm.herokuapp.com/node_latest_armhf.deb;
sudo dpkg -i node_latest_armhf.deb;

sudo npm install -g node-gyp;
